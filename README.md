# Internship Completion Analysis

## Project Overview

This project analyzes internship completion rates using Python, Pandas, and Matplotlib.

## Objectives

- Analyze internship enrollment
- Analyze completion rates
- Analyze dropout rates
- Compare departments
- Study mentor interaction
- Visualize results

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Files

- Internship_Analysis.ipynb
- internship_data.csv

## Results

- Calculated enrollment, completion, and dropout statistics.
- Created visualizations for department distribution and completion rates.
- Analyzed mentor interaction and internship duration.
